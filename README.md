## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Evelyn Araújo. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos específicos. A seguir, serão descritas as etapas do projeto:

### Etapa 1:
- Amazon S3 Intelligent-Tiering  
- Foco: Redução de custos de armazenamento  
- Descrição de caso de uso: Arquivos de dados são automaticamente movidos entre camadas para custo menor sem impactar no desempenho.

### Etapa 2:
- Amazon EC2 Spot Instances  
- Foco: Redução de custos computacionais  
- Descrição de caso de uso: Utilização de instâncias Spot para processamento de tarefas de baixa prioridade e ambientes de teste.

### Etapa 3:
- AWS Lambda  
- Foco: Computação serverless  
- Descrição de caso de uso: Funções serverless substituem servidores ociosos para tarefas periódicas e APIs simples.

## Conclusão
A implementação de ferramentas na empresa Abstergo Industries tem como esperado a redução de custos operacionais e maior eficiência nos processos de TI. Recomenda-se a continuidade da utilização das ferramentas implementadas e a busca por novas tecnologias que possam melhorar ainda mais os processos da empresa.
